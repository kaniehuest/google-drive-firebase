# Google Drive made with Firebase!

This "Google Drive" clone was made with:

- React
- Bootstrap
- FontAwesome
- Firebase


## Photos of the project
### Login
If you try to access the Dashboard without being logged, it'll redirect you to the login page.
![login](https://github.com/kaniehuest/Google-Drive-Firebase/blob/master/fotos/login.jpg?raw=true)

## Sign Up
You can change your password if you forgot, and you can go to the Login page with a single button
![SignUp](https://raw.githubusercontent.com/kaniehuest/Google-Drive-Firebase/master/fotos/register.jpg)

## Dashboard
